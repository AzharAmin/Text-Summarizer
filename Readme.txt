Automatic Text Summarizer Python based

In this min project my aim was to get summarized sentence from a large text file.
therefore, i wrote this code to get summarized sentence.
 
To avoid malfunctioning of code due to special characters or unnecessary spaces first i converted the long multi-lined text to one single line text with my code.

Then performed the particular operation:
like it will return the count of most used words and then will return a summarized text on word priority basis.
